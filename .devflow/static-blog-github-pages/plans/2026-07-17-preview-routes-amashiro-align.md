## 用户意图

分析 [amashiro.com](https://amashiro.com/) 的版式与交互，在**不改动正式站**的前提下，新开路由做视觉预览；满意后再迁移。

## 确认结论

1. **原站保留**：`BaseLayout` / 首页 / 博客 / 工具 零改动。
2. **预览命名空间**：`/preview/`（Pages 下为 `/mine-blog/preview/`）。
3. **四页**：首页、profile、gallery、contact。
4. **色系**：只消费现有 `tokens.css`（暗底 + `#39FF14`），不抄参考站蓝白皮。
5. **借鉴**：固定角菜单、竖排站名、展示型标题、分区档案、网格画廊、表单皮、品牌化 Footer。
6. **不做**：真实提交、对方素材/字体强依赖、本轮迁移切换。

## 成功标准

- 正式站视觉与路由行为不变。
- 预览四页可本地 `dev` / `build` 访问，互链完整，可一键回正式站。
- 气质可辨认为「暗色官方站壳 + 無限凜色」。
