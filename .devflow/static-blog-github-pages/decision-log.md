## 2026-07-15 · 技术栈

- **决策**：采用 **Astro（静态 SSG）+ 局部 React 岛屿（Islands）**
- **部署**：GitHub Pages 托管构建产物（非请求时 SSR）
- **原因**：
  - 用户希望接触新技术
  - 博客适合 Astro 内容集合与默认少 JS
  - 小工具可用 React 岛屿实现交互
  - 与 GitHub Pages 静态托管模型一致
- **明确不做（本阶段）**：
  - 不以 Next / Umi 做主栈
  - 不在 GitHub Pages 上做请求时 SSR
  - 不整站套 Ant Design 默认皮肤（避免与 DESIGN.md 气质冲突）
- **状态**：已生效
