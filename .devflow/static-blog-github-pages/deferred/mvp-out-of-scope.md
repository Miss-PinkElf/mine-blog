# 延期 · MVP 明确非目标

> 来源：Align / proposal 范围边界。第一版（骨架 MVP）只交付博客+单工具+Pages 链路。

## 内容与导航

| 项 | 说明 |
| --- | --- |
| 标签 / 分类归档 | 文章多了再做 |
| 关于页 | 品牌故事可后续补 |
| 站内搜索 | 文章量小不需要 |
| RSS | 订阅源后续 |
| 评论系统 | 需第三方或后端，与纯静态策略冲突时需另案 |

## 体验

| 项 | 说明 |
| --- | --- |
| 亮色 / 主题切换 | 当前锁定暗色 + 赛博绿 |
| 导航/UI 全繁体 | 已定：品牌名繁体「無限凜」，界面默认简体 |
| 格纹等角色装饰纹理 | 色板有黑绿格纹，MVP 未落地为纹理 |

## 工具与技术

| 项 | 说明 |
| --- | --- |
| 多个小工具 | MVP 仅 JSON 格式化；后续按卡片列表扩展 |
| Ant Design 局部引入 | 仅当重表单工具出现时再评估 |
| 请求时 SSR / API Routes | GitHub Pages 不支持；保持 SSG |
| 自定义域名 | 部署稳定后再配 |
| JSON 工具持久化（localStorage） | MVP 刷新清空可接受 |
